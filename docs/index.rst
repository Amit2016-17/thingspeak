Python binding for ThingSpeak API
=================================

`ThingSpeak <https://thingspeak.com>`_ is an open source “Internet of Things” application and API to store and retrieve data from things using HTTP over the Internet or via a Local Area Network. With ThingSpeak, you can create sensor logging applications, location tracking applications, and a social network of things with status updates.  https://github.com/iobridge/ThingSpeak

This repository is contains Python module that helps in talking to ThingSpeak API.
